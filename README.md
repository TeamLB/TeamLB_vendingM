#ウェブサーバー、データベースと連動するJAVA基盤の自動販売機を具現化
	
##*Enviroment & Requirement*
 * Apache 2.2.15
 * Mysql 14.14
 * PHP 5.5.32  

##*サービス*  
 * 実際の自動販売機の機能
 * データベースと連動して在庫を管理
 * ウェブで在庫管理をする機能
 * 販売量をグラフに提供する
